# Phase 06 Plan — Obsidian Vault Integration

## Goal
Replace plain `knowledge/` folder with Obsidian vault. Write properly linked markdown files with `[[double bracket links]]`, YAML frontmatter, tags, and backlinks. Enable visual knowledge graph via Obsidian.

## Steps

### 06-01: ObsidianStore Class
- [ ] Create `src/storage/obsidian_store.py`
- [ ] Extend/replace MarkdownStore interface
- [ ] `save_linked(item, related_content_ids)` — writes markdown with `[[links]]` to related content
- [ ] `_build_frontmatter(item, tags)` — YAML frontmatter with source, date, tags
- [ ] `_build_body(item, linked_topics)` — markdown body with `[[topic links]]`
- [ ] `create_topic_index(topic_slug, content_ids)` — creates topic overview note linking all content
- [ ] Keep MarkdownStore as fallback / for backwards compat

### 06-02: Backlinks Index
- [ ] Create `src/storage/backlinks_index.py`
- [ ] Track which content links TO each note (reverse link index)
- [ ] SQLite table: `backlinks(source_id, target_id, link_text)`
- [ ] `add_backlink(source_id, target_id, link_text)`
- [ ] `get_backlinks(content_id)` — returns list of content that links to this
- [ ] Update backlinks when saving new content with links

### 06-03: Obsidian Config + Daily Notes
- [ ] Add `OBSIDIAN_VAULT_PATH` to `config.py` (default: `knowledge/vault`)
- [ ] `create_daily_note(date, summary, content_ids)` — creates `YYYY-MM-DD.md` session note
- [ ] Session note links to all content consumed that day
- [ ] Add `![[embed]]` syntax for embedded related content

### 06-04: Integration with Existing Pipeline
- [ ] Update `sync_manager.py` to use ObsidianStore instead of MarkdownStore
- [ ] Update `intake_handler.py` to write to Obsidian vault
- [ ] Add `[[related topics]]` links based on embedding similarity

### 06-05: Tests
- [ ] `tests/test_obsidian_store.py` — test save, links, backlinks
- [ ] `tests/test_backlinks_index.py` — test backlink tracking

## Success Criteria
- Content saved to Obsidian vault with proper `[[links]]` and YAML frontmatter
- Topic index notes created for each topic folder
- Backlinks tracked and queryable
- Daily session notes created (research logging)
- Obsidian can open vault and show knowledge graph
