# Phase 05 Plan — yt-dlp YouTube Search

## Goal
Build yt-dlp YouTube search skill that scrapes YouTube metadata (title, views, channel, duration, URL) for zero-cost content discovery. Complements Tavily with YouTube-specific results.

## Steps

### 05-01: yt-dlp Search Handler
- [ ] Create `src/agent/handlers/ytdlp_search_handler.py`
- [ ] Implement `search_youtube(query, max_results=10)` — calls `yt-dlp --dump-json --search` and parses results
- [ ] Implement `get_video_metadata(url)` — get metadata for single video
- [ ] Return structured list: `[{title, url, uploader, view_count, duration, upload_date}]`
- [ ] Add `YTDLP_SEARCH_TRIGGER_PATTERNS` for auto-detection ("find videos on", "search youtube for", etc.)
- [ ] Add `is_youtube_search_query()` detection

### 05-02: yt-dlp Research Integration
- [ ] Extend `research_handler.py` to detect yt-dlp queries
- [ ] Add `handle_youtube_search()` path (separate from Tavily)
- [ ] When YouTube results found, offer: "Add video to knowledge base" / "Research more on this topic via Tavily"
- [ ] Selected YouTube URLs feed into existing `intake_handler.py` pipeline

### 05-03: Tests
- [ ] `tests/test_ytdlp_search.py` — mock subprocess, test metadata parsing
- [ ] Test `is_youtube_search_query()` detection patterns
- [ ] Test `search_youtube()` result structure

## Success Criteria
- yt-dlp search returns ranked YouTube results with metadata
- Auto-detected when user says "find videos on X" or "search YouTube for X"
- Selected videos can be added to knowledge base via intake pipeline
- No API key required
